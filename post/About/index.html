🏠 关于本站 / About this website

本站始建于2023年01月7日，意在取代QQ空间、微信朋友圈，用于登载一些文摘杂谈、技术分享、学习资料，偶尔也会分享一些摄影作品。
The website was built on January 7, 2023, intended to replace QQ Space and WeChat Friends Circle, and used to post some abstract talks, technology sharing, learning materials, and occasionally share some photography works.

此博客站点基于开源项目Gridea与开源项目托管平台GitHub创建。GitHub在国内访问并不稳定，出现网页加载速度慢、无法打开网页等问题均属正常现象，建议使用加速器加速GitHub。
This blog site is based on the Gridea open source project and GitHub open source project hosting platform. GitHub in China is not stable. It is normal to have problems such as slow web page loading and inability to open web pages. It is recommended to use an accelerator to accelerate GitHub.
